1. Какой файл должен быть основным для отображения сайта на GitHub Pages, и почему?
Ответ: Основным файлом должен быть "index.html", так как GitHub Pages использует его как точку входа

2. Какую видимость репозитория следует выбрать, чтобы сайт был доступен через GitHub Pages?
Ответ: Следует выбрать публичную видимость репозитория ("Public")

3. Можно ли настроить собственный домен для сайта на GitHub Pages? Если да, то где это сделать?
Ответ: Да, можно, это делается в разделе "Pages" настроек репозитория

4. Какие преимущества даѐт использование Git для работы с репозиторием вместо ручной загрузки файлов?
Ответ: Использование Git позволяет удобно обновлять сайт, фиксировать изменения и работать с локальной копией репозитория
